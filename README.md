AI-Based Virtual Gesture Controller | Software Project

Project Overview
This project presents an AI-based gesture recognition system that enables users to control computer functions without physical input devices. The system uses computer vision and hand tracking to interpret hand gestures and perform actions such as cursor movement, clicking, scrolling, typing, and volume control.

Features

- Hand gesture based virtual mouse
- Contactless virtual keyboard for typing
- Gesture based volume control
- Real-time hand tracking using MediaPipe
- Interactive graphical interface for selecting functions

Technologies Used
Python
OpenCV
MediaPipe
PyAutoGUI
Tkinter

Working Principle
The webcam captures real-time video frames, which are processed using OpenCV. MediaPipe detects hand landmarks and identifies finger positions. Based on gesture patterns, the system performs actions such as moving the cursor, clicking, scrolling, typing on a virtual keyboard, or adjusting system volume.

Applications

- Touchless human computer interaction
- Assistive technology for accessibility
- Smart environments and automation systems
- Hygiene sensitive environments such as hospitals
